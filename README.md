EcoStay Explorer 🌿
EcoStay Explorer is a modern, responsive web application built for sustainable travelers to discover and book eco-friendly accommodations worldwide. This project was developed as part of my internship to demonstrate skills in React, Tailwind CSS, and Client-Side Routing.

🚀 Key Features
Dynamic UI: Listing cards are rendered data-driven, making it easy to manage and update accommodations.

Seamless Navigation: Built with react-router-dom for a smooth Single-Page Application (SPA) experience.

Modern Styling: Fully styled with Tailwind CSS for a professional, responsive, and mobile-friendly design.

Interactive Components: Includes hover effects, responsive grids, and clean component architecture.

🛠 Tech Stack
Frontend: React, Vite

Styling: Tailwind CSS

Routing: React Router DOM

Development Tools: Git, GitHub, VS Code
